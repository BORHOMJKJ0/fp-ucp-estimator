# Frontend — Chat Estimation Interface

هذا المجلد يحتوي على الواجهة الأمامية لتطبيق تقدير المشاريع.

## ما تقدمه هذه الواجهة

الواجهة تسمح للمستخدم بـ:

- إدخال مفتاح OpenRouter API.
- بدء محادثة تفاعلية مع بوت التقدير.
- الإجابة على أسئلة متسلسلة حول المشروع.
- مشاهدة نتائج التقدير بطريقة منظمة.
- تنزيل تقرير PDF بمجرد اكتمال البيانات.

## تشغيل الواجهة

1. افتح الطرفية (Terminal).
2. انتقل إلى مجلد `frontend`:

```bash
cd frontend
```

3. ثبت الحزم:

```bash
npm install
```

4. شغل الواجهة:

```bash
npm run dev
```

5. افتح المتصفح وانتقل إلى العنوان الذي يظهر في الطرفية، عادة:

```bash
http://localhost:5173
```

## الأوامر الأساسية

- `npm run dev` : تشغيل الواجهة في وضع التطوير.
- `npm run build` : بناء نسخة نهائية للنشر.
- `npm run preview` : معاينة النسخة المبنية.

## كيف ترتبط الواجهة بالخادم

الواجهة تعتمد على الخادم الخلفي في مجلد `backend` لتلقي نتائج الذكاء الاصطناعي وتوليد PDF.
الطلبات تُرسل إلى واجهات برمجة التطبيقات التالية:

- `POST /api/chat`
- `POST /api/generate-pdf`

## للمستخدم غير المبرمج

لا تحتاج لتعديل الكود.

فقط:

1. شغل الخادم الخلفي في `backend`.
2. شغل الواجهة في `frontend`.
3. افتح التطبيق في المتصفح.
4. أدخل مفتاح OpenRouter.
5. تعاون مع البوت للحصول على التقديرات.

## ملفات مهمة

- `src/ChatApp.jsx` : منظم المحادثة، معالجة الرسائل، وحفظ مفتاح API.
- `src/App.jsx` : يعيد تصدير المكون الرئيسي.
- `vite.config.js` : إعدادات تشغيل Vite.

## ملاحظة

تأكد من تشغيل الخادم الخلفي قبل استخدام الواجهة حتى تعمل المحادثة وتوليد التقرير بشكل صحيح.

# Backend — Estimation Server

هذا المجلد يحتوي على الخادم الخلفي للمشروع.

## ما يفعله هذا الخادم

- يستقبل رسائل المستخدم من واجهة الدردشة.
- يرسل هذه الرسائل إلى OpenRouter API للحصول على ردود الذكاء الاصطناعي.
- يجمّع البيانات ويحسب التقديرات باستخدام طريقتين:
  - Function Point
  - Use Case Point
- يولد تقرير PDF جاهز للتحميل.

## ملفات مهمة

- `server.js` : الملف الرئيسي الذي يشغل الخادم.
- `package.json` : يحتوي على الإعدادات والأوامر.
- `fonts/NotoNaskhArabic-Regular.ttf` : خط عربي يستخدم لتوليد ملفات PDF باللغة العربية.

## كيف تبدأ الخادم

1. افتح نافذة الطرفية (Terminal).
2. انتقل إلى مجلد `backend`:

```bash
cd backend
```

3. ثبت الحزم:

```bash
npm install
```

4. شغل الخادم:

```bash
npm start
```

سيعمل الخادم عادة على:

```
http://localhost:5000
```

## كيف تعمل الواجهة مع الخادم

الواجهة الأمامية ترسل طلبات إلى هذه العناوين:

- `POST /api/chat`
  - لإرسال نص المحادثة ومفتاح OpenRouter.
  - يستقبل الرد من OpenRouter ويعيده إلى الواجهة.
- `POST /api/generate-pdf`
  - لإرسال بيانات المشروع المكتملة.
  - يولد ملف PDF ويعيده للواجهة حتى يتم تنزيله.

## متطلبات إضافية

- يجب أن يكون لديك مفتاح OpenRouter API صالح.
- لا تحتاج لإعداد ملف `.env` لأن المفتاح يُدخل من الواجهة الأمامية ويُرسل مباشرة.

## رسالة إلى غير المبرمجين

إذا كنت تستخدم التطبيق ولا تعرف البرمجة:

- لا تحتاج أن تغير أي كود.
- فقط شغل الخادم ثم الواجهة الأمامية.
- أدخل مفتاح OpenRouter في الواجهة.
- أجب على أسئلة البوت.
- اضغط زر "تحميل تقرير PDF" للحصول على الملف.

# Software Project Estimation Assistant

هذا المشروع هو أداة ذكاء اصطناعي تساعدك على تقدير حجم، جهد، وتكلفة المشاريع البرمجية بطريقة بسيطة.

## ما هو هذا المشروع؟

المشروع يتكون من:

- واجهة أمامية (`frontend`) تعرض ChatBot تفاعلي.
- خادم خلفي (`backend`) يتواصل مع OpenRouter API ويولد تقرير PDF.

المستخدم يكتب معلومات المشروع ويتلقى أسئلة مرتبة، ثم يحصل على تقديرين:

- تقدير بـ Function Point
- تقدير بـ Use Case Point

ويستطيع تنزيل التقرير بصيغة PDF.

## من يجب أن يستخدم هذا المستند؟

هذا المستند مخصص لأي شخص يريد فهم المشروع من دون خبرة برمجية كبيرة.

## كيف يعمل المشروع؟

1. المستخدم يفتح الواجهة الأمامية في المتصفح.
2. يدخل مفتاح OpenRouter API في مكانه المخصص.
3. يضغط "ابدأ التقدير".
4. يجيب على أسئلة البوت.
5. يحصل على نتائج التقدير ويستطيع تنزيل تقرير PDF.

## المحتوى العام للمجلدات

- `frontend/` : الواجهة الأمامية التي يراها المستخدم.
- `backend/` : الخادم الذي ينفذ الاتصالات ويولد ملف PDF.

## كيف تبدأ المشروع

### 1. تشغيل الخادم الخلفي

```bash
cd backend
npm install
npm start
```

### 2. تشغيل الواجهة الأمامية

في نافذة طرفية جديدة:

```bash
cd frontend
npm install
npm run dev
```

ثم افتح المتصفح وانتقل إلى العنوان الذي يعرضه Vite، عادةً:

```
http://localhost:5173
```

## ماذا تحتاج قبل التشغيل

- تثبيت Node.js (الإصدار 18 أو أحدث)
- مفتاح OpenRouter API صالح

## لماذا هذا المشروع مفيد؟

- يساعد على تقدير الوقت والتكلفة لمشروع برمجي.
- يقدم نتائج منظمة ويمكن تنزيلها عبر تقرير PDF.
- مناسب للمستخدمين الذين لا يعرفون تفاصيل التقدير الفني.

## ملاحظات مهمة

- إذا ظهرت رسالة تطلب مفتاح API، فتأكد أنك أدخلته بشكل صحيح.
- إذا كان التقرير PDF يظهر نصوصًا عربية بشكل غير صحيح، فهذا غالباً موضوع تنسيق ويمكن إصلاحه من خلال الخط في `backend/fonts`.
- لا يوجد إعداد خاص في المجلد الرئيسي، العمل يتم في `frontend` و `backend`.
